# Alpha-Blogy

> The application allows users to log in and write articles and edit them as well as viewing other people's articles and more.

![screenshot](./app/assets/images/alpha.png)

## Built With

- Ruby
- Ruby on rails
- bootstrap
- postgresql

### ruby on rails 6

### Install yarn

### Run rails s
